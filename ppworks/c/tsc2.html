for文を使って求める

for文を使って求める

    まず、n番目の仕事を請け負う人が何番目の人かを求めます。問題の入力例の場合、

    1番目 → 0
    2番目 → 1
    3番目 → 2
    4番目 → 3
    5番目 → 0
    6番目 → 1
    7番目 → 2

    で、3番目の人が請け負うことになります。ゼロから数えるので 1 を引いて 2 となります。

    3 を超えたら 0 に戻す方法は、文を使って、カウントを 0 に初期化すれば 0 〜 3 の範囲をループするようになります。
