Eメールアドレス

  • 例１

    • s = input()
t = input()
print(s + '@' + t)

  • 例２

    • s = input()
t = input()

mail_address = s + '@' + t
print(mail_address)

    文字列と文字列の間に指定の文字列を挟むという問題です。文字列と文字列の接続は + の記号を使います。

    s = input()
t = input()

mail_address = f"{s}@{t}"
print(mail_address)

    という書き方もあります。f文字列と呼ばれています。f は format (書式) の頭文字です。簡単に解説することは難しいのでここではしません。興味ある方はChatGPTなどに訊いてみてください。🤓

FINAL問題 N倍の文字列

N倍の文字列

  • 例１

    • N = int(input())
for _ in range(N):
    print('*', end='')
print()

  • 例２

    • N = int(input())
print('*' * N)

    Python3 ならば例２の書き方で出来あがってしまいます。文字列を整数で掛け算すると整数の回数分、同じ文字列が繰り返されて繋がります。この方法は、まぁまぁ使います。😁
